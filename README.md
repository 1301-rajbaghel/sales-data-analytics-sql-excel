# Sales Data Analytics using SQL and Excel

## Project Overview

This project analyzes sales data to identify revenue trends, category performance, customer purchasing behavior, and order status distribution.

The project demonstrates practical data analytics skills including data cleaning, SQL querying, dashboard creation, and business insight generation.

## Tools Used

- Microsoft Excel
- SQL
- Data Cleaning
- Pivot Tables
- Data Visualization

## Dataset

The dataset contains:

- Sales Orders
- Product Categories
- Revenue Information
- Order Status
- State-wise Sales Data

## Key Analysis Performed

### Revenue by State
Identified top-performing states contributing maximum revenue.

### Revenue by Category
Compared product categories based on total revenue generated.

### Order Status Analysis
Analyzed order completion and shipment performance.

## Key Insights

- Maharashtra generated the highest revenue.
- Set category contributed the highest sales.
- Most orders were successfully shipped and delivered.
- Revenue distribution varies significantly across states.
